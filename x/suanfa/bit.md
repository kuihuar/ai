### 更复杂的运算
1. 将x最右侧的n位清零， x&(~0<<n)
2. 获取x的第n位值（0或者1）， (x>>n)&1
3. 获取x的第n位的幂值， x&(1<<n)
4. 仅将第n位置为1， x|(1<<n)
5. 仅将第n位置为0， x&(~(1<<n))
6. 将x最高位至第n位（含）清零， x&((1<<n)-1)
7. 将第n位至第0位（含）清零， x&(~((1<<(n+1))-1))


### 统计二进制中1的个数

```go
func CountBits(n int) []int {
	ans := make([]int, n+1)
	for i := 1; i <= n; i++ {
		ans[i] = ans[i&(i-1)] + 1
	}
	return ans
}
```
#### 核心技巧
1. x&(x-1) 清零最低位的1
2. i & (i - 1) 的作用：去掉 i 的二进制表示中最右边的 1
#### 动态规划思想
1. 定义状态：dp[i] 表示数字 i 的二进制表示中 1 的个数
2. 状态转移方程：dp[i] = dp[i & (i - 1)] + 1
3. 初始状态：dp[0] = 0
4. 最终结果：dp[n]
5. i 的二进制中 1 的个数 = i & (i - 1) 的 1 的个数 + 1（因为 i & (i - 1) 比 i 少一个 1）
