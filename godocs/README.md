## 项目概览

这是一个基于 **GoFrame v2** 框架的 Go Web 应用项目，采用标准的 MVC 架构：

- **框架**: GoFrame v2.9.7
- **语言**: Go 1.23.0
- **架构**: MVC (Model-View-Controller)
- **部署**: Docker + Kubernetes

### 核心特性

1. **分层架构**: API → Controller → Service → DAO → Model
2. **版本管理**: API 支持版本控制（如 v1）
3. **中间件支持**: 统一的响应处理中间件
4. **配置管理**: YAML 配置文件
5. **容器化部署**: Docker + Kubernetes 支持

### 当前功能

- Hello World API: `GET /hello`
- Swagger 文档: `/swagger`
- OpenAPI 规范: `/api.json`
