# Kubeflow 学习指南

## 📖 什么是 Kubeflow？

Kubeflow 是一个运行在 Kubernetes 之上的机器学习平台，目标是让模型训练、超参搜索、流水线编排和在线推理具备可复现、可扩展、可运维的能力。

2. [安装与部署](./02-installation.md)
3. [Pipeline 实战示例](./03-pipeline-practice.md)

## 🎯 适用场景

- 需要统一管理数据科学与 MLOps 工作流
- 训练任务需要在 Kubernetes 上弹性调度
- 希望把实验、训练、发布流程标准化
