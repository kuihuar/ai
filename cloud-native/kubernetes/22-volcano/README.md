# Volcano 学习指南

## 📖 什么是 Volcano？

Volcano 是云原生批处理与 AI 作业调度平台，常用于大规模离线计算、分布式训练和高性能计算（HPC）场景。

## 🧭 学习路径

2. [安装与部署](./02-installation.md)
3. [批任务与队列实战](./03-job-practice.md)

## 🎯 为什么需要 Volcano？

- 原生 Kubernetes 对 Gang Scheduling 支持有限
- 批任务需要更细粒度的队列、配额和抢占策略
- AI/HPC 任务通常要求任务组整体调度与更高吞吐
